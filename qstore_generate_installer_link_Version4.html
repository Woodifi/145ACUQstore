Installation Backup Generator v3.0
📋 How to Use
  1. Export your backup: Open the Q-Store app, go to Settings (CO/OC only), click "Export Local (JSON)", and save the file.
  2. Copy the JSON: Open the exported file with a text editor, select all the text, and copy it.
  3. Paste it below: Paste the JSON into the text area below.
  4. Generate file: Click "Generate Installation File".
  5. Share the file: Send the downloaded `.qstore` file to your team members via email or file sharing.
  6. Users install: Users save the file and drag it onto the Q-Store app to install.
