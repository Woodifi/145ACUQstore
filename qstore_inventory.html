User Account Management
Accessible by CO/OC and Quartermaster roles only. PINs are hashed before storage — they cannot be retrieved, only reset. All login/logout events are recorded in the Audit Log.
⚠ CO/OC — Emergency Access Controls
These controls allow the CO/OC to reset access for any user or lock the entire system. Use in the event of a security incident, personnel change, or lost PIN. All actions are logged.
PIN reset sets all non-CO accounts to PIN 0000. Users must change their PIN on next login via their profile. Force sign-out invalidates all active sessions immediately.
